shadcn/ui (Neutral Theme)

  • Grey scale: Tailwind's "neutral" ‚Äî slate-50 (#f8fafc) through slate-900 (#0f172a)
  • CSS variables: --background, --foreground, --muted, --card, --border
  • Border radius: 0.5rem default (minimal)
  • Shadows: Very subtle or none
  • Color system: HSL-based for flexibility
  • Philosophy: Extremely minimal ‚Äî letting content speak

Design Principles from Both

  • White/light grey backgrounds ‚Äî body: #f5f7fb or #fafafa
  • Subtle borders over heavy shadows ‚Äî rgba borders, not drop shadows
  • Professional muted blues ‚Äî not vibrant indigo/teal
  • Small border-radius ‚Äî 3‚Äì6px, not 14‚Äì20px
  • System fonts or Source Sans Pro ‚Äî readable at 14‚Äì15px
  • Consistent grey text hierarchy ‚Äî primary: #1a1a2e, secondary: #6b7280, muted: #9aa0ac
