                          MODULE ARCHITECTURE
================================================================================

Interface Hierarchy:
  HRISConnector (abstract)
    └── BambooHRConnector (REST API implementation)
    └── CustomDBConnector (SQL Database implementation)

Data Model Relationships:
  Employee (with manager_id reference)
    → LeaveBalance (by employee_id)
    → LeaveRequest (by employee_id)
    → BenefitsPlan (by employee_id)
    → OrgNode (recursive hierarchy via manager_id)

Exception Hierarchy:
  ConnectorError (base)
    ├── ConnectionError
    ├── AuthenticationError
    ├── NotFoundError
    └── RateLimitError

Registry Pattern:
  ConnectorRegistry
    └── register(name, connector_class)
    └── get(name) -> connector_class
    └── list_connectors() -> [names]
