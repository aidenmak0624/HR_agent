## Troubleshooting

### Can't Login
1. Check your email address
2. Verify caps lock is off
3. Reset password: Click "Forgot Password" on login page
4. Check if account is active (contact HR if unsure)
5. Clear browser cookies and try again

### Leave Request Won't Submit
1. Ensure both dates are selected
2. Verify you have available balance
3. Check that dates are working days
4. Ensure you've filled all required fields
5. Contact HR if issue persists

### Can't See Team Members
1. Verify you're a manager in the system
2. Check that employees are assigned to you
3. Ensure they're marked as "active" status
4. Try refreshing the page
5. Clear browser cache

### Notifications Not Appearing
1. Check notification settings (Settings â†’ Notifications)
2. Allow browser notifications when prompted
3. Verify you haven't disabled the notification type
4. Try logging out and back in
5. Check your email if not receiving in-app

### AI Chat Not Responding
1. Check internet connection
2. Try asking a different question
3. Refresh the page
4. Clear browser cache
5. Contact IT support if problem persists

### Can't Download Documents
1. Check browser's download permissions
2. Ensure you have write access to downloads folder
3. Try a different browser
4. Check file naming (remove special characters)
5. Contact IT if issue persists

## FAQ

**Q: How far in advance should I request leave?**
A: At least 2 weeks preferred, but submit as soon as you know.

**Q: Can I cancel a leave request?**
A: Yes, if it's pending. Click the request and select "Cancel".

**Q: What if my manager rejects my leave request?**
A: The rejection reason will be shown. Discuss with your manager about alternative dates.

**Q: Can I check my colleague's leave schedule?**
A: Only your manager can see your team's schedule. Company calendar shows department-level info.

**Q: Where do I find company policies?**
A: Click "Documents" to view handbook and policies.

**Q: Is my chat conversation with the AI private?**
A: Yes, it's visible only to you and stored securely.

**Q: Can I change my email address?**
A: Contact HR admin. This affects your login credentials.

**Q: How do I export my data?**
A: Settings â†’ "Data Export" â†’ Select data type â†’ Download

**Q: How long are my documents stored?**
A: Generated documents are available for 90 days, then archived.

**Q: What happens to my data if I leave the company?**
A: Kept for 7 years per legal requirements. Can request deletion per CCPA.
